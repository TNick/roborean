# Changelog

## [Unreleased]

### Added

- OpenAPI-derived TypeScript types and a thin `fetch` client for the
  Roborean HTTP API.
- Client methods `updateProject` and `deleteProject` for project CRUD.
- Client methods `listRuns`, `previewDocument`, `downloadArtifact`, and
  `artifactDownloadUrl` for runs and document preview.
- Client methods `getTemplateContent`, `putTemplateContent`, and
  `deleteTemplateContent` for project template file CRUD.
- Client methods `listTemplateLibrary`, `getTemplateLibraryEntry`, and
  `getTemplateLibraryContent` for the global templates catalog; recipe
  entries include required bit type ids with display names.
